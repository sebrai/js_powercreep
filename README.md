# js_powercreep

**this is a js version of powercreep:**
&emsp; [pygame version](https://github.com/sebrai/powercreep "original version")

**version: beta 1.0.0**
**realesed: 13/3/2026**

## details:

- movement based survival game 
- smooth player control
- multiple abilities(*"specials"*) for you to choose from
- coins randomly plased on the map for players to collect
- dodge obstacles

## gameplay settings

- player color change 
- game difficulty slider (*1-5*)
- speacial selection
